
EN version 
**************** Creation & Install data base ***************************
To implement the database, you need several basic components: 
- you need to have downloaded the latest version of php https://www.php.net/downloads.php
- in the php.ini file, you need to "uncomment" the : 
extension=pdo_mysql
- you also need a php and mysql server with Docker or Uwamp to generate the database

-------------------------------------------------------------------------

Once the base components have been installed, you can download the db_passion.sql file from this folder and run it on the phpMyAdmin server.
